## Benjamin Lannon | August 6, 2019 | Careers in Code

---

# First off: What is Open Source?

Develop software where the code is public and free to obtain, modify, and even redistribute.

Examples include VS Code, NodeJS, Blender, Firefox

---

# The Licensing

[MIT](https://opensource.org/licenses/MIT), Apache, GPL, etc

Gives you certain rights. For the most part, what it does not offer is a warranty on said software.

---

# What projects do I contribute to?

- Syracuse.io: source for https://syracuse.io meetups site
- Gatsby: Performance focused React Static Site Generator

---

# Getting into it: The Code

- "Good First Issue" or "Help Wanted": Issues looking for new contributors. Usually very willing to help guide you to solve the issue.

---

# Getting into it: Everything else

- Docs
- A11y improvements
- Marketing / Design
- Promoting on social media

---

# Key Insights

- Look for something you care about
- Have good communication
- Have patience
- Don't be afraid to ask for help
